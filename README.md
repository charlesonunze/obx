# obx

## Prerequisites

Make sure you have [Docker](https://docs.docker.com/desktop/install/mac-install/) installed

## Run Locally

Clone the project

```bash
git clone https://github.com/charlesonunze/obx
```

Go to the project directory

```bash
cd obx
```

Start the server

```bash
make up
```

Run the test producer

```bash
node build/client.js
```
